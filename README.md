# CyberSec Trial (Windows)

This repository contains **distribution-only artifacts** for the CyberSec Trial.
It intentionally contains **no source code**.

## Download

- `cybersec-trial-setup.exe` (Windows installer)
- `cybersec-trial-setup.exe.sha256` (SHA256 checksum)

## Install

1. Run `cybersec-trial-setup.exe`
2. Launch **CyberSec Trial**
3. Activate using the Trial Key in `TRIAL_KEY.txt`

## Trial

- Trial duration: **48 hours** (starts at activation time)
